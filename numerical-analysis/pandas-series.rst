*****************
Pandas ``Series``
*****************


Creating
========
* 1-dimentional data structure similar to ``ndarray``

.. code-block:: python

    values = [1, 3, 5, np.nan, 6, 8]

    pd.Series(values)
    # 0    1.0
    # 1    3.0
    # 2    5.0
    # 3    NaN
    # 4    6.0
    # 5    8.0
    # dtype: float64


Indexes
=======
.. code-block:: python

    values = [1, 3, 5, np.nan, 6, 8]
    indexes = pd.date_range('1970-01-01', periods=6)
    # DatetimeIndex(['1970-01-01',
    #                '1970-01-02',
    #                '1970-01-03',
    #                '1970-01-04',
    #                '1970-01-05',
    #                '1970-01-06'], dtype='datetime64[ns]', freq='D')

    s = pd.Series(values, index=indexes)
    # 1970-01-01    1.0
    # 1970-01-02    3.0
    # 1970-01-03    5.0
    # 1970-01-04    NaN
    # 1970-01-05    6.0
    # 1970-01-06    8.0
    # Freq: D, dtype: float64

.. code-block:: python

    values = np.random.randn(5)
    indexes = ['a', 'b', 'c', 'd', 'e']

    s = pd.Series(values, index=indexes)
    # a    1.016521
    # b   -0.441865
    # c    0.519119
    # d    0.948774
    # e    0.207670
    # dtype: float64

Add values
----------
* Uses inner join
* ``fill_value``: If data in both corresponding Series locations is missing the result will be missing

.. code-block:: python

    import numpy as np

    a = pd.Series([1, 1, 1, np.nan], index=['a', 'b', 'c', 'd'])
    # a    1.0
    # b    1.0
    # c    1.0
    # d    NaN
    # dtype: float64

    b = pd.Series([1, np.nan, 1, np.nan], index=['a', 'b', 'x', 'y'])
    # a    1.0
    # b    NaN
    # x    1.0
    # y    NaN
    # dtype: float64

    a + b
    # a    2.0
    # b    NaN
    # c    NaN
    # x    NaN
    # y    NaN
    # dtype: float64

.. code-block:: python

    import numpy as np

    a = pd.Series([1, 1, 1, np.nan], index=['a', 'b', 'c', 'd'])
    # a    1.0
    # b    1.0
    # c    1.0
    # d    NaN
    # dtype: float64

    b = pd.Series([1, np.nan, 1, np.nan], index=['a', 'b', 'x', 'y'])
    # a    1.0
    # b    NaN
    # x    1.0
    # y    NaN
    # dtype: float64

    a.add(b, fill_value=0)
    # a    2.0
    # b    1.0
    # c    1.0
    # x    1.0
    # y    NaN
    # dtype: float64

.. code-block:: python

    s + s
    # a    2.033042
    # b   -0.883730
    # c    1.038237
    # d    1.897547
    # e    0.415339
    # dtype: float64
